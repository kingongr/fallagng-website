# Majesty Font Setup

## Steps to Add Majesty Font:

1. **Download Majesty font from Canva:**
   - Go to Canva and download the Majesty font files
   - You'll typically get files like: `Majesty-Regular.woff2`, `Majesty-Bold.woff2`, etc.

2. **Place font files here:**
   - Copy the Majesty font files (`.woff2` or `.ttf` format) to this `/public/fonts/` directory
   - Recommended files:
     - `Majesty-Regular.woff2` (or `.ttf`)
     - `Majesty-Bold.woff2` (if available)

3. **Font files will be automatically loaded** - The app is configured to use fonts from this directory.

## Supported Formats:
- `.woff2` (preferred - best compression)
- `.woff`
- `.ttf`

## Note:
If your font files have different names, update the font paths in `app/layout.tsx`.
